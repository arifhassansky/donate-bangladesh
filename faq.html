Frequently Asked Questions

1. What is the document object model (DOM)?

The Document Object Model (DOM) is a tree-like structure representing a web page, allowing JavaScript to manipulate HTML elements, attributes, and content dynamically.

2. How do you select an element from a dom?

You can select elements from the DOM using methods like document.getElementById('id'), document.querySelector('selector'), document.getElementsByClassName('className'), or document.getElementsByTagName('tag').

3. What is the event delegation in the context of the DOM and why is it useful?

Event delegation is a technique where a parent element handles events for its children through event bubbling. It's useful for improving performance by reducing event listeners and managing dynamically added elements, making the code more efficient and maintainable.

4. How do you manipulate an element's attribute and styles using the DOM ?

You can manipulate an element's attributes using setAttribute() or getAttribute(), and styles by directly modifying the style property, like element.style.color = "red"; for dynamic styling changes.
